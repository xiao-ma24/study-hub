| 项目 | 内容 |

## 重点章节

### 🔴 必考（占分60%+）

1. **树与二叉树**
   - 二叉树遍历（前/中/后/层序）
   - BST的插入与删除
   - 哈夫曼树构造

2. **排序算法**
   - 快速排序（过程 + 代码）
   - 归并排序
   - 堆排序
   - 各排序算法的时间/空间复杂度比较

3. **图**
   - DFS/BFS遍历
   - 最小生成树（Prim/Kruskal）
   - 最短路径（Dijkstra）

4. **查找**
   - 二分查找
   - 哈希表（冲突解决方法）

### 🟡 重要

5. 线性表（顺序表/链表操作）
6. 栈与队列（应用：表达式求值、括号匹配）

---

## 避坑指南

::: warning 过来人经验
1. **一定要画图**。树的遍历、图的搜索，不画图很容易绕晕。
2. **手写代码练习**。考试要手写伪代码，平时只看不练到考场会写不出来。
3. **复杂度必须背**。时间/空间复杂度是选择填空必考，不能丢分。
4. **排序算法对比表要熟**。快排、归并、堆排的复杂度和稳定性是高频考点。
:::

---

## 复习建议

```
第1步：过一遍知识点框架，理解每种结构的操作（2-3天）
  ↓
第2步：手写练习核心算法代码（快速排序/二叉树遍历/Dijkstra）（3-4天）
  ↓
第3步：刷历年真题，熟悉题型（2-3天）
  ↓
第4步：模拟考试，限时手写代码（1-2天）
```
