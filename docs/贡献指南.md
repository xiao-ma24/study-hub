# 贡献指南

> 感谢你有兴趣为学习资料库贡献内容！以下是几种贡献方式。

---

## 📤 方式一：直接上传文件（最简单，推荐）

### 第一步：准备文件

把你的资料文件（PDF / Word / 图片）按以下目录结构放好：

```
docs/public/files/科目名/类型/文件名.pdf
```

**示例**：
```
docs/public/files/高等数学/真题/2024秋-期末.pdf
docs/public/files/高等数学/笔记/期末复习提纲.pdf
docs/public/files/数据结构/重点/老师画重点.docx
```

### 第二步：更新科目主页的表格

打开对应科目的 `index.md`（如 `docs/高等数学/index.md`），在对应分区的表格里加一行，格式如下：

```markdown
| [文件名.pdf](/files/高等数学/真题/文件名.pdf) | 2024秋 | ✅ | [下载](#) / [在线预览](#) |
```

> 💡 链接路径说明：`/files/` 对应 `docs/public/files/` 目录，VitePress 会自动把这个目录下的文件打包到网站中。

### 第三步：提交

- **如果你有 GitHub 权限**：直接 push 到仓库
- **如果没有**：把文件和要更新的信息发到 QQ 群，管理员帮你上传

---

## 📋 文件命名规范

| 资料类型 | 推荐命名 | 示例 |
|---------|---------|------|
| 复习笔记 | `内容描述.pdf` | `期末复习提纲.pdf` |
| 历年真题 | `年份学期-考试类型.pdf` | `2024秋-期末.pdf` |
| 老师重点 | `内容描述.pdf` | `老师画重点.pdf` |
| Word 笔记 | `内容描述.docx` | `知识点汇总笔记.docx` |

---

## 📋 表格格式模板

每次添加资料，在对应科目的 `index.md` 里找到对应分区的表格，加一行：

**笔记类**：
```markdown
| [文件名.pdf](/files/科目名/笔记/文件名.pdf) | PDF | — | 2024-12 | [下载](#) / [在线预览](#) |
```

**真题类**：
```markdown
| [文件名.pdf](/files/科目名/真题/文件名.pdf) | 2024秋 | ✅ | [下载](#) / [在线预览](#) |
```

> 📌 暂时先用 `#` 占位下载/预览链接，后续会加上真正的预览功能。现阶段点击 PDF 链接浏览器会自动打开或下载。

---

## 📤 方式二：Fork + Pull Request（适合有 GitHub 经验的贡献者）

1. Fork 本仓库
2. 按上述步骤添加文件和更新表格
3. 提交 Pull Request

---

## 📋 资料质量要求

::: tip 提交前检查
- [ ] 内容准确，没有明显错误
- [ ] 文件名清晰，能看出内容
- [ ] 标注资料来源（整理人/回忆版/原创等）
- [ ] 真题标注年份和学期
- [ ] 不涉及老师明确要求保密的内容
:::
