# 大学英语 — 学习路线

## 基本信息

| 项目 | 内容 |
|------|------|
| 先修课程 | 无 |
| 建议复习时长 | 1周 |
| 难度 | ⭐⭐ |
| 特点 | 以课本内容为主，难度低于四六级 |

---

## 考试题型（通用）

| 题型 | 占比 | 复习重点 |
|------|------|---------|
| 听力 | 20-30% | 课后听力材料 |
| 阅读理解 | 30-40% | 课本文章 + 真题阅读 |
| 词汇与语法 | 15-20% | 课后单词、语法点 |
| 翻译 | 10-15% | 课文重点句子 |
| 写作 | 15-20% | 背模板 |

---

## 复习策略

1. **背单词**：课后单词表是重点，考试直接考
2. **读课文**：翻译题和阅读题很多来自课文
3. **背作文模板**：考前背2-3个万能模板
4. **刷真题**：了解题型和难度
