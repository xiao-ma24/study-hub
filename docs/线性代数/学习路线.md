# 线性代数 — 学习路线

## 基本信息

| 项目 | 内容 |
|------|------|
| 先修课程 | 无 |
| 建议复习时长 | 1-2周 |
| 难度 | ⭐⭐⭐ |
| 特点 | 概念抽象，计算套路固定 |

---

## 重点章节

### 🔴 核心必考

1. **行列式计算**
   - 降阶法、性质化简
   - 克莱姆法则

2. **矩阵运算**
   - 乘法、逆矩阵、转置
   - 初等变换求逆

3. **线性方程组**
   - 齐次/非齐次方程组求解
   - 解的结构（基础解系）

4. **特征值与特征向量**
   - 求特征值和特征向量
   - 矩阵对角化

### 🟡 重要

5. 向量组的线性相关性
6. 二次型及其标准形

---

## 避坑指南

::: warning 注意
1. **计算容易出错**。线代的计算步骤多，一步错步步错，建议每步检查。
2. **概念要理解透**。线性相关/无关、基础解系这些概念是解题的基础。
3. **初等变换要熟练**。求逆矩阵、解方程组、化简二次型都要用，是核心技能。
:::
