# 关于

## 项目简介

**学习资料库**是一个学院复习资料共享平台，旨在打破期末信息差，让每个同学都能高效获取复习资料。

## 为什么做这个项目

大学里，信息差是造成成绩差距的重要原因。同样一门课，有人有历年真题和重点笔记，有人只有课本。这个项目想把散落在QQ群、人脉中的资料集中起来，变成一个结构化、可搜索的在线知识库。

## 技术栈

- **框架**：[VitePress](https://vitepress.dev/) — 基于 Vue 的静态网站生成器
- **部署**：GitHub Pages — 免费托管
- **自动化**：GitHub Actions — push即自动构建部署
- **内容**：Markdown — 简单易写的格式
