## 第一章：极限与连续

### 1.1 极限的计算方法

| 方法 | 适用场景 | 关键点 |
|------|---------|--------|
| 洛必达法则 | 0/0 或 ∞/∞ 型 | 确保是未定式再用 |
| 等价无穷小替换 | 乘除法中的无穷小 | 加减法慎用 |
| 重要极限 | 含 sinx/x 或 (1+1/x)ˣ | 记住两个基本形式 |
| 夹逼准则 | 不易直接求的极限 | 找两个夹的函数 |

### 常用等价无穷小（x→0时）

```
x ~ sin x ~ tan x ~ arcsin x ~ arctan x ~ ln(1+x) ~ eˣ - 1
1 - cos x ~ x²/2
(1+x)ᵃ - 1 ~ ax
```

**注意**：等价无穷小替换只适用于乘除，加减法中慎用（仅在整体为因式乘积时可替换）。

### 1.2 连续性与间断点

- **连续**：lim(x→a) f(x) = f(a)
- **间断点分类**：
  - 第一类：左右极限都存在（可去型、跳跃型）
  - 第二类：至少一侧极限不存在（无穷型、振荡型）

## 第二章：导数与微分

### 2.1 求导法则速查

| 法则 | 公式 |
|------|------|
| 乘法法则 | (uv)' = u'v + uv' |
| 除法法则 | (u/v)' = (u'v - uv') / v² |
| 链式法则 | [f(g(x))]' = f'(g(x)) · g'(x) |

### 2.2 高频求导题型

1. **复合函数求导**：层层剥开，逐层求导
2. **隐函数求导**：对方程两边求导，解出 dy/dx
3. **参数方程求导**：dy/dx = (dy/dt) / (dx/dt)
4. **对数求导法**：适用于幂指函数 y = u(x)ᵛ⁽ˣ⁾

## 第三章：中值定理与导数应用

### 3.1 三大中值定理

> **证明题套路**

1. **罗尔定理**：f(a)=f(b) → 存在 ξ 使 f'(ξ)=0
2. **拉格朗日中值定理**：存在 ξ 使 f'(ξ) = [f(b)-f(a)]/(b-a)
3. **柯西中值定理**：两个函数的拉格朗日

### 3.2 导数应用

| 概念 | 判断方法 |
|------|---------|
| 单调性 | f'(x)>0 递增，f'(x)<0 递减 |
| 极值 | f'(x₀)=0 且 f''(x₀)≠0；f''(x₀)>0→极小值，f''(x₀)<0→极大值 |
| 凹凸性 | f''(x)>0 凹，f''(x)<0 凸 |
| 渐近线 | 水平(lim f(x))、垂直(无定义点)、斜(多项式除法) |

## 第四章：积分

### 4.1 不定积分方法

| 方法 | 适用 | 关键 |
|------|------|------|
| 第一换元（凑微分） | 被积函数含复合结构 | 熟记微分公式 |
| 第二换元 | 含根号 | 三角换元、根式换元 |
| 分部积分 | 两类函数乘积 | ∫udv = uv - ∫vdu |

### 分部积分选择口诀

> **反对幂三指**（优先级：反三角函数 > 对数 > 幂函数 > 三角函数 > 指数函数）
>
> 排在**前面**的当 u（求导），排在**后面**的当 dv（积分）

### 4.2 定积分重要性质

- **变上限积分求导**：d/dx ∫ₐˣ f(t)dt = f(x)
- **牛顿-莱布尼茨公式**：∫ₐᵇ f(x)dx = F(b) - F(a)

## 第五章：多元函数微积分

### 5.1 偏导数

- 对 x 求偏导时，把 y 视为常数
- 二阶偏导：连续时可交换求导顺序

### 5.2 二重积分

**直角坐标**：∬_D f(x,y) dxdy

**极坐标**（当积分区域为圆/扇形时优先）：
```
x = r cosθ,  y = r sinθ
dxdy = r dr dθ
```
