## 重点章节排序

> 按"考试占比 × 掌握难度"排序，越靠前越重要

### 🔴 必须掌握（占分60%+）

1. **极限与连续**
   - 重点：洛必达法则、等价无穷小替换、重要极限
   - 题型：求极限（几乎每年必考）

2. **导数与微分**
   - 重点：复合函数求导、隐函数求导、参数方程求导
   - 题型：计算题

3. **中值定理与导数应用**
   - 重点：罗尔定理、拉格朗日中值定理、单调性与极值
   - 题型：证明题（难点）

4. **不定积分与定积分**
   - 重点：换元积分法、分部积分法、变上限积分求导
   - 题型：计算题（分值大）

### 🟡 重要（占分25%左右）

5. **多元函数微分学**
   - 偏导数、全微分、方向导数

6. **重积分**
   - 二重积分计算（直角坐标 + 极坐标）

### 🟢 了解即可（占分15%左右）

7. 级数（收敛判别、幂级数展开）
8. 微分方程（一阶线性、可分离变量）

## 避坑指南

::: warning 过来人的教训
1. **别只看课本**。课本的例题和考试题型差距很大，一定要找历年真题。
2. **别从头看到尾**。按上面的优先级来，先把前4章吃透，后面有时间再看。
3. **计算能力是王道**。高数考试计算量很大，建议每天练5道积分题保持手感。
4. **中值定理证明题别放弃**。虽然难，但每年必考一道，背几个套路模板就能拿分。
:::

## 推荐复习顺序

```
第1步：看学习路线，了解重点分布（10分钟）
  ↓
第2步：看复习提纲，过一遍知识点框架（1-2小时）
  ↓
第3步：刷近3年真题，摸清题型（3-4小时）
  ↓
第4步：针对薄弱章节，看笔记细啃（按需）
  ↓
第5步：二刷真题，模拟考试（2-3小时）
  ↓
第6步：看老师重点，查漏补缺（30分钟）
```

## 常见问题

**Q：高数上和高数下哪个难？**
A：普遍认为高数下更难（多元函数、级数、微分方程），但高数上的计算量更大。建议上的基础打好，下的学习会更轻松。

**Q：需不需要买辅导书？**
A：有历年真题就够了。辅导书可以作为补充，但优先级低于真题和笔记。

**Q：挂科率高吗？**
A：因学校而异，但高数是大学挂科率最高的课之一。认真刷真题基本能过，想拿高分需要把证明题也拿下。
