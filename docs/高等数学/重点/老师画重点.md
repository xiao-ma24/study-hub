## 老师明确说的重点

### 🔴 一定会考

1. **求极限** — 洛必达 + 等价无穷小
   - 老师原话："这种题必考，不会的趁早练"

2. **隐函数求导** — 老师特意强调了两遍

3. **分部积分** — 老师说"计算题里肯定有一道"

4. **中值定理证明** — 罗尔定理或拉格朗日
   - 老师原话："证明题就考中值定理，背几个套路就行"

5. **二重积分** — 老师说"极坐标和直角坐标都可能考"

### 🟡 大概率考

6. 变上限积分求导
7. 函数的极值与最值
8. 一阶线性微分方程

### 🟢 说不准

9. 级数收敛判别（老师含糊其辞："看一下就好"）
10. 曲线的凹凸性与拐点

## 老师划的范围

::: tip 复习范围
- 第一章到第五章：**全考**
- 第六章（多元函数）：**重点考偏导数和二重积分**
- 第七章（级数）：**只考收敛判别，不考幂级数展开**
- 第八章（微分方程）：**只考一阶线性，不考二阶**
:::

---

## 老师给的建议

1. **"把课后习题做一遍"** — 但实际上考试题型和课后习题有差距，建议以真题为主
2. **"带计算器"** — 虽然高数计算大多不需要计算器，但带上以防万一
3. **"证明题别空着"** — 即使不会，写个罗尔定理的条件和结论也能拿步骤分
4. **"计算题要写出过程"** — 只有答案没有过程不给满分
