## 一、选择题（每题4分，共20分）

**1.** 极限 $\lim_{x \to 0} \frac{\sin 3x}{\sin 5x}$ 的值为（&nbsp;&nbsp;&nbsp;&nbsp;）

A. $\frac{3}{5}$ &emsp; B. $\frac{5}{3}$ &emsp; C. $1$ &emsp; D. $0$

> **答案：A**
> 解析：等价无穷小替换，sin3x ~ 3x，sin5x ~ 5x，所以 = 3/5

**2.** 设 $f(x) = x^3 - 3x$，则 f(x) 的极大值为（&nbsp;&nbsp;&nbsp;&nbsp;）

A. 0 &emsp; B. 2 &emsp; C. -2 &emsp; D. 4

> **答案：B**
> 解析：f'(x) = 3x² - 3 = 0 → x = ±1。f''(x) = 6x。x=-1时 f''(-1)=-6<0，极大值 f(-1) = -1+3 = 2

**3.** 下列函数在 x=0 处不连续的是（&nbsp;&nbsp;&nbsp;&nbsp;）

A. $f(x) = |x|$ &emsp; B. $f(x) = x\sin\frac{1}{x}$（定义f(0)=0）
C. $f(x) = \frac{\sin x}{x}$（定义f(0)=1） &emsp; D. $f(x) = \frac{1}{x}$

> **答案：D**

**4.** 定积分 $\int_0^1 x e^x dx$ 的值为（&nbsp;&nbsp;&nbsp;&nbsp;）

A. 0 &emsp; B. 1 &emsp; C. e-1 &emsp; D. e

> **答案：B**
> 解析：分部积分 = [xe^x]₀¹ - ∫₀¹ e^x dx = e - (e-1) = 1

**5.** 级数 $\sum_{n=1}^{\infty} \frac{1}{n^2}$ 的收敛性为（&nbsp;&nbsp;&nbsp;&nbsp;）

A. 发散 &emsp; B. 条件收敛 &emsp; C. 绝对收敛 &emsp; D. 无法判断

> **答案：C**
> 解析：p级数，p=2>1，绝对收敛

## 二、计算题（每题10分，共50分）

**6.** 求极限 $\lim_{x \to 0} \frac{e^{x^2} - 1 - x^2}{x^4}$

> **解析：**
> 洛必达法则（0/0型），连续求导：
> 
> 第一次：$\lim_{x \to 0} \frac{2x e^{x^2} - 2x}{4x^3} = \lim_{x \to 0} \frac{2x(e^{x^2}-1)}{4x^3}$
> 
> 等价无穷小 $e^{x^2}-1 \sim x^2$：
> 
> $= \lim_{x \to 0} \frac{2x \cdot x^2}{4x^3} = \frac{2}{4} = \frac{1}{2}$
> 
> **答案：1/2**

**7.** 设 $y = y(x)$ 由方程 $e^y + xy = e$ 确定，求 $y'(0)$ 及 $y''(0)$

> **解析：**
> x=0时，e^y = e → y(0) = 1
> 
> 对方程求导：$e^y \cdot y' + y + xy' = 0$
> 
> 代入 x=0, y=1：$e \cdot y'(0) + 1 = 0$ → **y'(0) = -1/e**
> 
> 再求导：$e^y(y')^2 + e^y y'' + 2y' + xy'' = 0$
> 
> 代入 x=0, y=1, y'=-1/e：$e \cdot \frac{1}{e^2} + e \cdot y''(0) - \frac{2}{e} = 0$
> 
> $y''(0) = \frac{1}{e^2}$

**8.** 计算不定积分 $\int \frac{1}{x^2 \sqrt{x^2+1}} dx$

> **解析：**
> 令 x = tanθ，dx = sec²θ dθ
> 
> $\int \frac{\sec^2\theta}{\tan^2\theta \cdot \sec\theta} d\theta = \int \frac{\cos\theta}{\sin^2\theta} d\theta = -\frac{1}{\sin\theta} + C$
> 
> 回代：sinθ = x/√(x²+1)
> 
> **答案：$-\frac{\sqrt{x^2+1}}{x} + C$**

**9.** 计算二重积分 $\iint_D (x+y) dxdy$，其中 D 由 y=x 和 y=x² 围成

> **解析：**
> 交点：x = x² → x=0 或 x=1
> 
> $\int_0^1 dx \int_{x^2}^{x} (x+y) dy$
> 
> $= \int_0^1 [xy + \frac{y^2}{2}]_{x^2}^{x} dx$
> 
> $= \int_0^1 (x^2 - x^3 + \frac{x^2}{2} - \frac{x^4}{2}) dx$
> 
> $= \frac{1}{3} - \frac{1}{4} + \frac{1}{6} - \frac{1}{10} = \frac{3}{20}$
> 
> **答案：3/20**

**10.** 解微分方程 $y' - \frac{1}{x}y = x^2$（x > 0）

> **解析：**
> 一阶线性微分方程，P(x) = -1/x
> 
> 积分因子：$\mu = e^{\int -\frac{1}{x}dx} = e^{-\ln x} = \frac{1}{x}$
> 
> $\frac{y}{x} = \int x dx = \frac{x^2}{2} + C$
> 
> **答案：$y = \frac{x^3}{2} + Cx$**

## 三、证明题（15分）

**11.** 证明：当 x > 0 时，$\ln(1+x) > x - \frac{x^2}{2}$

> **证明：**
> 令 $f(x) = \ln(1+x) - x + \frac{x^2}{2}$，x ≥ 0
> 
> f(0) = 0
> 
> $f'(x) = \frac{1}{1+x} - 1 + x = \frac{1 - (1+x) + x(1+x)}{1+x} = \frac{x^2}{1+x}$
> 
> 当 x > 0 时，f'(x) = x²/(1+x) > 0
> 
> 所以 f(x) 在 [0, +∞) 上单调递增
> 
> 当 x > 0 时，f(x) > f(0) = 0，即 $\ln(1+x) > x - \frac{x^2}{2}$
> 
> **证毕。**

## 四、应用题（15分）

**12.** 求由曲线 $y = x^2$ 与 $y = 2-x^2$ 围成的区域的面积，并求该区域绕 x 轴旋转所得旋转体的体积。

> **解析：**
> 
> 交点：x² = 2-x² → x = ±1
> 
> **面积：**
> $S = \int_{-1}^{1} [(2-x^2) - x^2] dx = \int_{-1}^{1} (2 - 2x^2) dx = [2x - \frac{2x^3}{3}]_{-1}^{1} = \frac{8}{3}$
> 
> **体积：**
> $V = \pi \int_{-1}^{1} [(2-x^2)^2 - x^4] dx = \pi \int_{-1}^{1} (4 - 4x^2) dx = \pi [4x - \frac{4x^3}{3}]_{-1}^{1} = \frac{16\pi}{3}$
> 
> **答案：面积 8/3，体积 16π/3**
